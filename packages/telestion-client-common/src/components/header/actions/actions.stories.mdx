### Custom Action

You can also define a custom action button.

To keep things consistent, please use an icon-only `<ActionButton />` from the Adobe Spectrum Design system and include
a tooltip:

<Canvas>
	<Story height={'128px'} name="Custom Action" args={{}} inline={false}>
		{args => (
			<CommonWrapper>
				<Header
					right={
						<Actions>
							<TooltipTrigger>
								<ActionButton onPress={args.onCustomActionClick}>
									<CustomIcon />
								</ActionButton>
								<Tooltip>Click me to achieve something</Tooltip>
							</TooltipTrigger>
						</Actions>
					}
				/>
			</CommonWrapper>
		)}
	</Story>
</Canvas>
