## Examples

<Canvas>
	<Story height={'64px'} name="Default logo" args={{}}>

You can customize the logo by passing `appLogo` to the `<AppLogoContext />` (or, more commonly, the `<CommonWrapper />`):

<Canvas>
	<Story height={'64px'} name="Custom logo" args={{ logo: newLogo }}>
		{args => (
			<AppLogoContext appLogo={args.logo}>
				<AppLogo />
			</AppLogoContext>
		)}
	</Story>
</Canvas>
