## Examples

### Default login form

Per default, all text fields begin empty:

<Canvas>

We can, however, add initial values for both the url (`initialServerURL`) and username (`initialUsername`)

<Canvas>

				<LoginPage>
					<LoginForm {...args} />

	</Story>
</Canvas>
