# Pages

The `<Pages />` wrapper "magically" does all the necessary routing, including authentication checks and redirects.

```
const SimplePage = () => {
    const history = useHistory();

    const handleClick = useCallback(event => {
        signIn(() => {
            history.push('/dashboard');
        });
        event.preventDefault();
    }, [history]);

    return (
        <>
            <h3>Login window</h3>

			<button onClick={handleClick}>Open Dashboard</button>
        </>
    );
};

LoginPage.routing = {
    type: 'default',
    path: '/',
    exact: true
};
```
