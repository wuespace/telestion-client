## Import

```tsx
import { TelestionClient } from '@wuespace/telestion-client-core';
```

## Examples

To get some data in these examples, we'll define a simple helper component:

```tsx
import { useTitle } from '@wuespace/telestion-client-core';

export function TitleUser() {
	const title = useTitle();

	return <p>Title: {title}</p>;
}
```
